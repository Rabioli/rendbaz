Webhosting

Provicionamiento de un espacio en el servidor (donde esta página esta alojada) dedicado a mostrar y guardar o sostener (hosting) tu sitio web.

Precio mensual: desde $120 mxn

Términos y condiciones:

  • 1GB-10GB Espacio en el disco
  • 10 GB - 100 GB Uso de datos (Las páginas web pesan KBs)
  • Asistencia en horas laborales
  • El tipo de contenido alojado no debe infringir leyes de Canadá, pues es donde se encuentra el servidor
  • Sitios web estáticos, es decir portales, no aplicaciones web con un gran uso de recursos.
